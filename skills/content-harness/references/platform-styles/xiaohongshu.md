# 小红书发布风格指南

> 从关注的创作者（赛博禅心等）的高质量笔记中提炼的发布规律。
> Stage 9 小红书发布流程的核心参考文档。

---

## 平台特性

- **核心内容形态：** 图文笔记（主流）/ 视频（暂不涉及）
- **封面决定点击率：** 封面图 + 标题是唯一"门面"，封面文字要清晰可读（与公众号不同）
- **算法偏好：** 互动率（收藏>点赞>评论）> 粉丝量；新号冷启动靠话题标签
- **读者行为：** 快速浏览，停留时间短；"收藏"代替阅读完（收藏率是核心指标）

---

## 内容风格规范

### 标题格式

```
[情绪/价值钩子] + [具体内容描述]
```

常见有效格式：
- **疑问式**：「用AI写文章，3个月涨粉10万？我来还原真相」
- **结果式**：「靠这个思路，我的公众号打开率从3%到18%」
- **反常识**：「不要每天发！小红书涨粉最快的人都在做这件事」
- **清单式**：「分享5个我在用的AI写作工具（附使用场景）」

**禁止：** 标题超过 20 字、纯概念词（"深度剖析""全方位解析"）

### 封面图规范（与公众号封面不同）

- **尺寸：** 3:4 竖版（1080×1440）为主流；也可 1:1（1080×1080）
- **文字必须出现在封面：** 标题或核心钩子，字号大、清晰
- **风格：** 干净背景 + 大字 / 生活感照片 + 文字叠加 / 拼图
- **颜色：** 暖色系（米白、莫兰迪色、浅橙）用于生活内容；深色/高对比用于科技内容

### 正文结构

```
开头句（钩子，1-2句）
↓
分点呈现（3-7个观点，每条不超过3句）
↓
收尾（邀请互动 或 行动号召，1-2句）
```

**字数：** 200-800 字最佳，避免长文；核心信息前置

**分点格式（选一种）：**
```
① 第一点
② 第二点
③ 第三点
```
或
```
▸ 关键词：内容描述
▸ 关键词：内容描述
```

**禁止：** emoji 堆砌（3个/段以内）、段落超过 150 字、无分点的大段文字

### 话题标签策略

每篇 3-5 个标签，结构：
```
#大类目(10亿+) #中类目(1亿+) #细分场景(100万级) #个人品牌词
```

AI内容常用组合：
```
#AI工具 #人工智能 #AI写作 #效率工具 #[具体工具名]
```

## 赛博禅心风格特征（初始观察）

> 待积累更多文章样本后完善

**初步特征：**
- 标题常用「我在用」「亲测」等第一人称，真实感强
- 正文结构清晰，每个要点用数字或 ▸ 标记
- 图片多为工具截图或概念示意图，不用纯装饰图
- 评论区积极互动，常置顶一条问题引导讨论

## Stage 9 小红书发布流程

### 前置条件

- 公众号版文章已完成（Stage 9 微信部分先跑）
- 用户在 Stage 7 确认了"发布到小红书"
- 已有 Stage 6 生成的小红书版内容草稿

### Step XHS-1: 内容转化（从公众号版改写）

读取 Stage 6 生成的小红书版草稿，按以下规则优化：

1. **标题改写**：公众号标题 → 小红书钩子格式（参考上方格式规范）
2. **正文压缩**：
   - 保留 3-5 个核心观点
   - 每个观点压缩到 1-3 句
   - 删除所有过渡性描述（"我认为"、"上文提到"等）
3. **格式适配**：添加数字编号或 ▸ 符号，确保可扫读
4. **收尾添加**：结尾加 1 句互动邀请（"你在用什么工具？评论区聊聊"）

### Step XHS-2: 图片准备

**封面图：**
- 使用 Stage 6.5 生成的封面（需改为竖版 3:4）
- 或新生成一张封面：`python3 "$SKILL_DIR/scripts/generate_cover.py" --prompt "PROMPT" --size 1080x1440 --out /tmp/xhs_cover_$(date +%Y%m%d).png`
- 封面必须包含文字（在 prompt 中指定文字内容）

**配图（可选）：**
- 最多 9 张（小红书图文上限）
- 优先复用公众号插图（需裁成 1:1 或 3:4）
- 每张图要有信息增量，纯装饰图跳过

### Step XHS-3: 话题标签生成

根据文章类型自动生成话题组合：
```python
# 文章类型 → 话题模板
article_type_tags = {
    "opinion": ["#AI观察", "#科技思考", "#人工智能"],
    "creator-share": ["#AI工具", "#效率工具", "#工具分享"],
    "how-to": ["#AI教程", "#操作教程", "#学习笔记"],
    "news-reaction": ["#科技资讯", "#AI动态", "#行业观察"],
}
# + 文章具体工具/话题词（从标题提取）
# + 个人品牌标签（如有）
```

### Step XHS-4: 发布（手动，不支持 CDP 自动化）

**小红书暂无可靠 API 发布通道。** 流程输出可直接粘贴的内容：

```
--- 小红书发布内容包 ---
【标题】
[标题文本]

【正文】
[正文文本]

【话题标签】
[标签列表]

【封面图路径】
[路径]

【配图路径列表】
[路径列表]
```

展示给用户，用户手动在小红书 APP 或网页版创作中心发布。

### Step XHS-5: 效果回收（发布后 24-72 小时）

提醒用户记录：点赞数、收藏数、评论数、涨粉数。
记录到 knowledge-base.md 的小红书效果区域，供下次发布参考。
