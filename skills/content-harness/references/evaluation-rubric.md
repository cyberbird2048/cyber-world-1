# 评估标准 — 基于标杆文章校准

## 标杆文章特征（这就是 9 分长什么样）

以下特征从 4 篇标杆文章中提取。评估时，对标这些特征打分。

### 特征 1: 递进式升维结构

每篇标杆文章都是"小切口进入 → 逐步升维 → 哲学/战略层面收束"：

- 《写日记》：旅行记录技巧 → 记忆与遗忘 → AI 时代的自我存在证明
- 《宠物系统》：CLI 小游戏 → 概率算法拆解 → "跟概率较劲是人类最古老的浪漫"
- 《女娲.skill》：同事 skill 梗 → 认知蒸馏方法论 → "女娲造的不是人，是镜子"
- 《Anthropic 疯狂一周》：产品发布时间线 → Agent OS 三层架构 → 围墙花园战略

**评估标准：** 文章是否有清晰的"从具体到抽象"的递进？结尾是否比开头高一个维度？

### 特征 2: 反直觉钩子

每篇标杆文章在前 3 句就给出一个让人意外的判断：

- "最危险的不是不会用 AI 的人"（否定常识）
- "我靠，我怎么什么都不记得了"（个人崩溃时刻）
- "为什么只蒸馏同事？你可以蒸馏芒格"（概念跳跃）
- "OpenClaw 被封杀 → 4 天后 Anthropic 自己做了一个"（戏剧性反转）

**评估标准：** 开头 3 句是否有"意外感"？读者是否有"必须读下去"的冲动？

### 特征 3: 具象化能力

每个抽象概念都配了具体类比或场景：

- "办了一张健身房年卡，然后每天带 18 个朋友去洗澡"（讽刺 API 滥用）
- "先请你吃饭，再卖你保险"（平台策略的本质）
- "白板废物仙人掌"（游戏化语言降低技术门槛）
- "你这辈子真正活过的证据"（日记的存在意义）

**评估标准：** 文章中每个核心观点是否都有具体的类比/场景/案例支撑？有几个"像 X 一样"式的表达？

### 特征 4: 信息密度

标杆文章没有"水段"。每一段删除后文章都会变差。

**反面案例（AI 常犯）：**
- 用两段说同一个观点的不同表述
- "总的来说…"段落（纯总结，无新信息）
- 引出下文的过渡段（"接下来我们看看…"）

**评估标准：** 逐段检查——这一段删掉后文章是否变差？如果不变差，这段是水。

### 特征 5: 自然语感（反 AI 感）

标杆文章读起来像"一个聪明朋友在跟你聊天"，不像机器生成：

**自然感来源：**
- 口语短句："别这样。"、"不是这样的。"
- 偶尔粗口或语气词："我靠"、"爽爆了"
- 不完整的句子、碎片式表达
- 第二人称直接对话："你可能觉得…但其实…"
- 情绪波动：从冷静分析突然切到感性感叹

**AI 感来源（禁止）：**
- "随着…的发展"、"在当今…时代"
- "值得注意的是"、"不可否认"
- "总的来说"、"综上所述"
- 每段结尾都有"因此…"式总结
- 过度使用 emoji 做视觉装饰
- 段落长度高度一致（真人写作段落长短不一）

### 特征 6: 一句话收尾

每篇标杆文章的最后一句都能脱离全文独立传播：

- "恰恰是那些你觉得太日常了的东西，才是你这辈子真正活过的证据"
- "跟概率较劲这件事，可能是人类最古老的浪漫之一"
- "女娲造的不是人。是镜子。"
- "围墙花园最怕的，是隔壁有人在建一个没有墙的花园"

**评估标准：** 最后一句话是否有"金句感"？是否重新定义了全文？能否单独发一条朋友圈？

---

## 评分标准

| 分数 | 含义 | 特征 |
|------|------|------|
| 9-10 | 标杆级 | 以上 6 个特征全部具备，文章有"只此一篇"的独特性 |
| 8-8.9 | 可发布 | 5 个以上特征具备，可能某个维度稍弱但整体扎实 |
| 7-7.9 | 需优化 | 3-4 个特征具备，有明显短板但骨架好 |
| 6-6.9 | 需重写 | 结构或观点有硬伤，修补不如重来 |
| <6 | 放弃 | 灵感本身可能需要重新提炼 |

## 终极校验：活人感终审（来自卡兹克四层自检体系）

所有维度打完分后，做最后一个判断：

> 整篇文章读下来，是否像"一个有见识的普通人在认真跟你聊天"，而非"AI在输出信息"？

如果答案是"像AI"，不管分数多高，都需要重写。这是最高优先级的否决条件。

活人感的具体检测点：
- 全文是否有至少一处**情绪波动**（从分析突然转到感叹、从冷静转到激动）
- 段落长度是否**参差不齐**（AI倾向于写等长段落）
- 是否有**不完整的句子或碎片式表达**
- 是否使用了**第一人称"我"的真实体验**
- 是否有至少一处**立场表达**（不是中立描述，而是"我认为"）

## 对抗式评估 Sub-Agent Prompt 模板

spawn 独立 Agent 时使用以下 prompt（填入实际内容）：

```
你是一个独立的内容质量审核员。你没有参与这篇文章的写作过程，你的任务是客观评估它。

## 用户的原始灵感
{{灵感原文}}

## 待评估的文章
{{完整文章}}

## 评估标准

好的 AI 领域文章有 6 个核心特征：

1. **递进式升维**：从具体/小的切入 → 逐步升维 → 哲学/战略层面收束
2. **反直觉钩子**：开头 3 句有"意外感"，让人必须读下去
3. **具象化**：每个抽象概念都配了具体类比或场景
4. **信息密度**：每一段删掉后文章都会变差，没有水段
5. **自然语感**：读起来像"一个聪明朋友在跟你聊天"，不像 AI 生成
6. **一句话收尾**：最后一句能脱离全文独立传播

## 请你做以下判断

1. **逐段审查**：每一段的信息增量是什么？有没有可以删掉不影响全文的段？
2. **活人感终审**：整篇读下来，像一个有见识的真人在跟你聊天，还是像 AI 在输出信息？说出具体依据。
3. **和标杆的差距**：对标以上 6 个特征，这篇文章在哪些地方做到了，哪些地方没到？具体到段落。
4. **最致命的 3 个问题**：按严重程度排序，每个问题给出具体修改建议。
5. **灵感还原度**：用户的原始灵感是否是文章的灵魂？还是被稀释成了背景？
6. **综合判定**：可发布 / 需优化 / 需重写

注意：不要客气，不要鼓励，直接说问题。宁可严格也不要放水。
```

## 规则扫描禁止词表

以下表达出现任何一个即触发 AI 感 FAIL：

- "随着…的发展"
- "在当今…时代"
- "值得注意的是"
- "不可否认"
- "总的来说"
- "综上所述"
- "毋庸置疑"
- "接下来我们看看"
- "让我们拭目以待"
- "深度解析"
- "全面盘点"

## 评估流程（对抗式）

1. **规则扫描**（主 Agent 自行执行）：8 项硬规则检查，输出 PASS/FAIL
2. **对抗评估**（spawn 独立 Agent）：使用上方 prompt 模板
3. 综合两层结果，决定：进入优化 / 进入多平台改写 / 回到结构重生成
