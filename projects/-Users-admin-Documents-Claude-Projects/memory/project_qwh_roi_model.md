千万会（Ten Million Club）VIP 体系方案，经 4.2 沟通版本后 ROI 模型做了重大修正。

## 核心决策（2026-04-13 确认）

**收入口径**：$1.5/笔（非交易额百分比），与平台实际收入率 1.15bps 交叉校验误差 8%。
**Why:** 原 0409 版用 0.5% 佣金率假设过高（实际 0.0115%），导致 ROI 虚高 5.9x。修正后纯增量 ROI 算不过来，需加入收益回馈。
**How to apply:** 所有收入计算统一用 交易笔数 × $1.5 × 7.8 汇率，不再用交易额×佣金率。

## 人群界定（新口径）

| 路径 | 条件 | 人数 |
|------|------|------|
| A（资产型） | 月均资产 ≥ $100万 USD | 434人 |
| B（交易型） | 年交易额 ≥ $1,000万 USD + 月均资产 ≥ $3万 USD | 2,245人 |
| A+B并集 | 去重 | **2,623人** |

## 关键数据

- A+B占平台 0.48% 用户，贡献 19.5% 资产 + 35.2% 交易额 + 13.4% 营收
- A路径434人占平台资产 13.9%；B路径2,245人占平台交易额 29.7%
- 1个千万会用户 = 32个普通用户
- 年交易笔数：817万笔
- 年交易收入：HKD 9,557万
- 人均年收入：A路径 HKD 19,677 / B路径 HKD 39,293 / 整体 HKD 36,434

## ROI 模型（两变量）

```
可接受成本 = VIP收入 ×（收入计提比例 + 交易增量率）
VIP年收入：HKD 9,557万
年预算：HKD ~1,318万（首年）
打平条件：计提 + 增量 ≥ 12.9%
推荐：10%计提 + 5%增量 = HKD 1,434万 ≥ 1,318万 ✓
```

**不再使用四杠杆（留/引/活/拓）框架。**

## 行业对标

- 互联网券商 VIP 运营费率：5-10%（富途/老虎）
- 高净值留存：15-25%（Schwab/HSBC）
- 千万会 12.9% 介于两者之间，作为高净值专项留存合理
